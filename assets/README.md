This folder holds local assets for the static site.

Place the hero background image here as `hero.jpg` so the site can reference it locally.

Original Unsplash image used previously:
https://images.unsplash.com/photo-1586023492125-27b2c045efd7?w=1920

How to download and add the image (macOS example):
1. Open the original URL in a browser.
2. Right-click the image and choose "Save Image As...".
3. Save it into this `assets/` folder and rename to `hero.jpg`.

After adding `assets/hero.jpg` the hero section will use the local file automatically.
